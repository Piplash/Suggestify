Acerca de Suggestify

Suggestify es una aplicación web diseñada para entregarte 5 recomendaciones musicales en base a una canción de tu elección. Te gustó una canción y quieres encontrar más similares? Prueba Suggestify!

¿Cómo funciona?

Para conseguir esto, nos conectamos a la API oficial de Spotify mediante el inicio de sesión del usuario, permitiéndonos en primer lugar ingresar una canción y obtener como respuesta una sugerencia de 5 canciones similares. Luego, con las sugerencias listas, puedes crear una playlist directo a tu cuenta con el nombre que quieras!

¿Puedo ver el código?

Sí! Este es más que nada un proyecto personal y sin fines más allá de la entretención. Puedes revisar el código en mi perfil de GitHub.

¿Se almacenan datos?

No! Este proyecto solo utiliza endpoints para entregar una canción a Spotify, recibir sugerencias y crear playlists. Para más detalles puedes leer la Política de Privacidad

About Suggestify

Suggestify is a webapp designed to suggest you 5 songs according to the one you enter. Loved a song and want to find more music like that? Try Suggestify!

How does it work?

We connect directly to the Official Spotify API through your login, allowing us to enter a song and get similar suggestions as a response. Then you can even create a playlist with the suggestions and name it as you please!

Can I see the code?

Sure! This is just a personal project for nothing but fun. You can check the code on my GitHub Profile..

Is data stored?

Nope! This project only uses endpoints to send a song to Spotify, get suggestions as a response and create a playlists. You can check the Privacy Policy for more details.
